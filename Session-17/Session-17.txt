1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζει ολόκληρο το url μας;
Δεν μπορούμε να κάνουμε ένα request χωρίς να ξέρει το API να γνωρίζει το URL. Πρέπει να γνωρίζει το URL για να μας στείλει τα δεδομένα από το ένα άκρο της σύνδεσης στο άλλο.

2. Αν κάνω ένα GET request και ζητήσω ένα jpeg image, το response τι content-type πρεπει να εχει;

 To response θα μας επιστραφεί:

Response.ContentType: image/jpeg

όπου το image είναι το είδος αρχείου και το jpeg είναι ο τύπος αρχείου (επέκταση).

3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να κρατηθεί στην cache για 1 μέρα;
Συνήθως τα GET REQUEST έχουν μικρή διάρκεια ζωής. Όμως θα μπορούσε από έναν server να έχει cache-control που διαρκεί όση ώρα του οριστεί.

4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests;
Θα μπορούσαμε να έχουμε ένα site για επίσκεψη μόνο και να βλέπαμε βασικά πράγματα χωρίς να κάναμε παραπάνω requests με τον server (π.χ. login, αγορές κ.α.).

5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests;
Ναι θα μπορούσε να υπάρχει ενας server που να δέχεται δεδομένα (POST) από χρήστες και να μην στέλνει τίποτα. 

6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα υποχρεωτικό πεδίο;
Αν δεν έχω στείλει ένα υποχρεωτικό πεδίο σημαίνει ότι είναι client based error και η βάση δεν θα το δεχτεί. Άρα το error θα είναι 400, Bad Request.


7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη βάση;
Αν δεν υπάρχει επικοινωνία με την βάση ενώ όλα στον client δουλεύουν σωστά (π.χ. εχει internet, οι μέθοδοι είναι σωστοί κτλ) τότε είναι αδυναμία επικοινωνίας, δηλαδή, server based error. Το error αυτό θα είναι 503, Service Unavailable. 

8. Μπορώ να κάνω Login με GET request;
Τις συνδέσεις τις κάνουμε πάντα με την μέθοδο POST για λόγους ασφάλειας. Σε περίπτωση που χρησιμοποιήσουμε GET θα πρέπει να στείλουμε τις παραμέτρους του URL πράγμα το οποίο θα αφήσει τα credentials  μας εκτεθειμένα.

9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST;
Η διαφορά είναι ότι  την POST την χρησιμοποιούμε για να κάνουμε create ένα resource ενώ την PUT για να κάνουμε update ένα resource.

10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα
αρχείο;
Μπορούμε να χρησιμοποιήσουμε και PUT και να κάνουμε συγχρόνως update το resource, αλλά σύμφωνα με την απάντηση της προηγούμενης ερώτησης προτιμότερο είναι να χρησιμοποιήσουμε την μέθοδο POST.
